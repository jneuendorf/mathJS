CONSTANTS Class

Defined in: doc_base/Constants.js:3
Module: mathJS

This is a class (not actually but logically) that contains all definitions of constants. Those constants can be con- or disjoined (see functions 'and' and 'or') but already con- or disjoined can't be con- or disjoined any further (so nesting is not supported).
